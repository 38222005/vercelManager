# TabBar 图标资源

请在此目录下放置以下TabBar图标：

## 首页图标
- `home.png` - 未选中状态（灰色 #999999）
- `home-active.png` - 选中状态（蓝色 #007AFF）

## 租赁图标
- `rental.png` - 未选中状态
- `rental-active.png` - 选中状态

## 维修图标
- `maintenance.png` - 未选中状态
- `maintenance-active.png` - 选中状态

## 新车图标
- `newVehicle.png` - 未选中状态
- `newVehicle-active.png` - 选中状态

## 其他图标
- `other.png` - 未选中状态
- `other-active.png` - 选中状态

## 图标规格
- 尺寸：81x81 像素（推荐）
- 格式：PNG（支持透明背景）
- 颜色：
  - 未选中：#999999
  - 选中：#007AFF

## 临时方案
如果没有图标资源，可以：
1. 使用图标字体库（如 iconfont）
2. 使用在线图标资源
3. 使用纯文本或emoji代替
4. 从图标网站下载（如 iconfont.cn、flaticon.com）
